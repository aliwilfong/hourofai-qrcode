# hourofai-qrcode
This repository is a static webpage for generating a qr code from a url or one line of text
there is a box for inputting the text and the generated qr code is shown below. A button,
underneath, lets you download the code as an PNG image. This code is generated by ai agent,
as an example of how to use it. 

Testing 
=======

Serve this folder with a simple HTTP server and open the page for testing. 
This repository is deployed as a static webpage using the default GitHub pages
workflow.
